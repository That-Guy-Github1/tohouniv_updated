# So basically the idea I had was making several event chains, which would have the following pattern
# 1. You select a location you want to go with your rival/lover
# 2. You select an activity which you think your rival/lover will enjoy
# 3. You have to pass some arbitary challenge. The player gets a randomly selected requirement they have to pass. Following ideas were collected:
#	- Challenge of Prestige: have at least x Prestige
#	- Challenge of Tradition: have at least x Tradition
#	- Challenge of Defenestration of your Lover: literally throw Kaguya/Mokou out of the window (basically a Danmaku check)
#		- This one is special: goal is here to FAIL the check to entertain the rival, but it must happen CLOSE.
#		- If you are too weak your rival is not entertained, if you are too strong you kill your rival, which definitely kills the mood too
#	- Challenge of War Crimes: scorch the earth of the province where the date is going on because ... reasons
#	- Challenge of Mana: have x mana, spend x mana
#	- Challenge of Men: have x manpower
#	- Challenge of Deity: have x as your deity
# 4. You get a score for completing a challenge and negative score for failing challenges
# 5. After around 3 challenges the date is finished
# 6. Depending on the score you get a bad, normal and good result
#
# Also originally there is a privilege which allows your rival to visit you whenever she wants but this has proven to be unfun (I think the code still is in there and should probably be deleted).
#
# The reason why I couldn't implement the ideas was not time but a lack of creativity of how the hell I should write it
# At this point of time, where you read the comments here, I has my 7th attempt on it (I deleted the previous versions because they were just embarassing - all what is left are the comments).
# I cannot write anything close to romantic event texts and concepts for the life of me, I am very sorry about it
# Sooooo... if somebody is willing to write the localization first then I am more than happy to script this
#
